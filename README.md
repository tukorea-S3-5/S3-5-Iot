# Smart Heart Rate Monitor Band

## 소개
ESP32와 MAX30102 센서를 이용한 심박 측정 스마트 밴드 프로토타입 테스트 환경입니다.

## 시작하기 전 필수 준비사항 (Prerequisites)
이 코드를 실행하려면 아두이노 IDE에서 아래 라이브러리를 반드시 설치해야 합니다.
* **라이브러리 Sketch 메뉴 -> Include Library -> Manage Libraies에서 검색 후 설치:**
    * `SparkFun MAX3010x Pulse and Proximity Sensor Library` (by SparkFun Electronics)
